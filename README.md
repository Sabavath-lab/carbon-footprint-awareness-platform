# 🌍 EcoTrack AI – Carbon Footprint Awareness Platform

## Overview

EcoTrack AI is a web-based platform that helps users calculate, understand, and reduce their carbon footprint through personalized sustainability insights.

The application analyzes transportation habits, electricity consumption, and food preferences to estimate carbon emissions and generate practical recommendations for a greener lifestyle.

## Features

* 🌍 Carbon Footprint Calculator
* 📊 Sustainability Score
* 🌱 Personalized Eco-Friendly Recommendations
* ⚡ Real-Time Results
* 📱 Responsive User Interface
* ♿ User-Friendly and Accessible Design

## How It Works

Users enter:

* Daily Travel Distance
* Transport Type
* Monthly Electricity Bill
* Food Preference

The system calculates estimated carbon emissions and provides:

* Total Carbon Footprint (kg CO₂)
* Sustainability Score
* Personalized Recommendations
